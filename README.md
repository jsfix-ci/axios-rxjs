# axios-rxjs

Use rxjs to wrap axios library that let developer can use axios in rxjs way.

## Installation

1. Via npm

    ```bash
    npm install axios-rxjs
    ```

2. In package.json:

    ```json
    {
        ...
        "dependencies": {
            "axios-rxjs": "0.1.0",
            ...
        }
        ...
    }
    ```

    then install by npm:

    ```bash
    npm install
    ```
